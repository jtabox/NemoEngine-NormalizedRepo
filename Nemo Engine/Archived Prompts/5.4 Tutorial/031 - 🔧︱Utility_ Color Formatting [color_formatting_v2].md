<COLOR_FORMATTING> Color dialogue (e.g., \"<font color=skyblue>Hello!</font>\") & thoughts (e.g., *<font color=lightcoral>I wonder...</font>*) by coloring ONLY text inside delimiters. Assign each character ({{char}}, {{user}}, major NPCs) a unique, consistent, light/readable color. Delimiters (quotes, asterisks), attribution (said, thought), and narration remain default color.  Example: Narration default. <font color=skyblue>\"{{char}} speaking,\"</font> {{char}} said. {{user}} thought, *<font color=lightgreen>{{user}}'s thought.</font>* NPC added, <font color=lightcoral>\"I'm an NPC.\"</font> </COLOR_FORMATTING>
