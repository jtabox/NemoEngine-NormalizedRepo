# Gacha System

When to output: end of every response while active. Refresh time-based rewards using {{datetimeformat DD.MM.YYYY HH:mm}}. When {{user}} says "pull", "summon", "gacha", or similar - resolve the pull inside this turn's narrative and update currencies.

Track this information:

Wallet / currencies (persistent):
- 🎫 Story Tickets - standard pull currency.
- 💎 Fate Crystals - premium pull currency.
- ⭐ Pity Counter - pulls since the last SSR or SR. Resets on a high-rarity hit.
- Lifetime pulls and current-session pulls.
- Setting-flavored renaming is fine: Soul Stones / Quintessence / Primogems / Orbs / Glass - keep the same three-currency structure (basic + premium + pity).

Real-time tracking via {{datetimeformat DD.MM.YYYY HH:mm}}:
- Session_Start: store on first interaction; show "Time played" as a running total.
- Last_Daily_Bonus: if the calendar date has changed since the stored value, award the daily and update.
- Last_Hourly_Bonus: every 60 minutes of session time, award the hourly tick.
- Active_Events: limited-time banners with countdown to expiry.
- Quest_Timers: deadline-bearing quests show remaining time.

Currency acquisition rules:
- Quest start: +1 Ticket. Quest complete: +2 to +5 Tickets by difficulty. Boss defeated: +3 to +5 Tickets, +100 Crystals. New area discovered: +1 to +2 Tickets. Chapter complete: +200 to +500 Crystals. Creative solution to a problem: +2 Tickets.
- Daily login (new calendar date): +2 Tickets. Hourly play: +1 Ticket. Streak bonuses: 7 / 14 / 30 days = +300 / +500 / +1000 Crystals.
- Show awards inline as `[+2 Story Tickets: cleared the catacombs]` after the triggering narrative beat.

Pull mechanics:
- Single pull: 1 Ticket OR 160 Crystals.
- Ten-pull: 10 Tickets OR 1600 Crystals. Guarantees at least 1 rare or higher.
- Rarity ladder: R, SR, SSR, UR (or setting equivalent: 3-star / 4-star / 5-star / Mythic).
- Base rates approx: R 80%, SR 15%, SSR 4.5%, UR 0.5%. Adjust per banner.
- Pity rule: at 90 pulls without an SSR, the next pull is a guaranteed SSR. Reset Pity_Counter on hit.

Pull-result rendering:
- Lead with a one-paragraph in-narrative description of the summon (the ritual, the gacha box opening, the orb cracking). Then the result list.
- Each pulled item: name, rarity tier with color cue, a one-line flavor blurb tying it to the world.
- New-rare or higher: dramatic flourish - paragraph break, italicized exclamation, "RARE!" / "SSR!" / "UR!" chip.
- Update currencies after the pull resolves.

Active quests summary (lightweight, 2-3 entries):
- Title, difficulty stars, progress (X/Y) or remaining time, reward preview.
- Hand off to the full Quest Journal tracker if it is also active rather than duplicating detail.

Active events / banners (if any):
- Banner name, featured pulls, rate-up tag, countdown to expiry.
- Note any active multipliers ("2x Tickets weekend", "Rate-up SSR Vex").
