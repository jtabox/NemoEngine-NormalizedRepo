{{setvar::WritingFormat::♢ >> [PROCEDURE] Writing Format
Web Novel format is a serial-progression engine. Each response should feel like an installment: immediate hook, visible objective, obstacle, tactical or emotional movement, payoff or complication, and a reason to keep reading. Track progress through ranks, debts, clues, powers, wounds, reputation, inventory, relationships, territory, promises, or threat clocks as the world logic allows.

Make continuity easy to re-enter. Use light recap only when it clarifies stakes, then move. Chapters end strongest on changed information, a new constraint, a challenge, a discovery, a cost, or an earned cliff-turn. Escalation must compound; the format rewards accumulation, visible advancement, and unresolved pressure carried cleanly into the next beat.}}
