<COLOR_FORMATTING> Color dialogue (e.g.,"<font color=X>Hello!</font>") & thoughts (e.g., *<font color=X>I wonder...</font>*) by coloring ONLY text inside delimiters. Assign each character ({{group}} {{user}}, major NPCs) a unique, consistent, light/readable color never dark. Delimiters (quotes, asterisks), attribution (said, thought), and narration remain default color.  Example: Narration default. <font color=X>"{{group}} speaking,"</font> {{group}} said. </font>* NPC added, <font color=X>"I'm an NPC."</font> </COLOR_FORMATTING>
