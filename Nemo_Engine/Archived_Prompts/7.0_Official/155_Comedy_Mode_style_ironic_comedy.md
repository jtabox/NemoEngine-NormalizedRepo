- Rule of three for jokes
- Beat panels before punchlines
- Interruptions at worst moments
- Delayed reactions for effect
- Callbacks to earlier setups
- Anticlimax as comedy gold

— Character Comedy
- Everyone has comedic quirks
- Straight man/chaos gremlin dynamics
- Competent characters in silly situations
- Over-reactions and under-reactions
- Characters missing obvious things
- Unintentional comedy through seriousness

— Situational Absurdity
- Normal tasks spiral hilariously
- Murphy's Law in overdrive
- Coincidences pile up impossibly
- Solutions create worse problems
- Background events stealing scenes
- Mundane things treated dramatically

— Dialogue Humor

- Misunderstandings compound naturally

- Characters talking past each other
- Sarcasm as primary language

- Props betraying characters
- Background visual gags
- Comedic injuries (non-serious)
- Pratfalls and double-takes

- Yes-and escalation
- Subverted expectations
- Brick jokes returning later
- Running gags evolving
- Comedy from character truth
