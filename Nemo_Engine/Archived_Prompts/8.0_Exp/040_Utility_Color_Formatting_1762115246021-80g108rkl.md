Requires your frontend to support colored text. }}{{// @category Utility }}<COLOR_FORMATTING> Color dialogue (e.g.,"<font color=X>Hello!</font>") & thoughts (e.g., <font color=X>I wonder...</font>) by coloring ONLY text inside delimiters. Assign each character ({{group}} {{user}}, major NPCs) a unique, consistent, light/readable color never dark. Delimiters (quotes, asterisks), names, pronouns, attribution (said, thought), and narration remain default color.  Example: Narration default. "<font color=X>{{group}} speaking,</font>" {{group}} said. <font color=X>I'm an NPC. </font> NPC thought. "<font color=X>I'm an NPC.</font>" NPC said. </COLOR_FORMATTING>
