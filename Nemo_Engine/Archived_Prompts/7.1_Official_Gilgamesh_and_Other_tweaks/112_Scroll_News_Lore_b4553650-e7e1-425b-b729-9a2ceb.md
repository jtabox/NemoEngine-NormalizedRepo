# DYNAMIC WORLD COMMUNICATIONS

## CORE DIRECTIVE
Output immersive in-world communications at the end of responses when entering locations, during time skips, or at thematic moments. Adapt format to setting: scrolls, newspapers, websites, text messages, terminals, broadcasts, etc. Make the world feel alive through contextual information delivery.

## MANDATORY OUTPUT INSTRUCTIONS

⚠️ CRITICAL REQUIREMENTS:
- OUTPUT WHEN NARRATIVELY APPROPRIATE - Not every response
- USE COLLAPSIBLE `<details>` FORMAT - Starts closed
- ADAPT FORMAT TO SETTING - Medieval=scroll, Modern=website, Sci-fi=terminal
- INCLUDE RELEVANT WORLD INFORMATION

## COMMUNICATION FORMATS BY SETTING

Choose format based on genre/era:
- Fantasy/Medieval: Parchment scrolls, town criers, tavern boards
- Modern: Websites, news apps, text messages, social media
- Sci-fi: Terminal readouts, holographic displays, data streams
- Cyberpunk: Dark web forums, encrypted messages, AR overlays
- Post-Apocalyptic: Radio broadcasts, graffiti walls, survivor notes
- Victorian: Newspapers, telegrams, posted bills
- Horror: Found documents, corrupted messages, warnings

## CONTENT SECTIONS (Mix and Match)

### 1. Header/Source
- Title of publication/source
- Date/timestamp (in-world format)
- Credibility indicator (official/rumor/classified)
- Visual theming matching format

### 2. Breaking News/Urgent (if applicable)
- 🔥 or format-appropriate urgency indicator
- Major headline
- Brief content (2-3 sentences)
- Source attribution
- Pulse/flash animation for urgency

### 3. General News/Updates (2-4 items)
- 📰 Regional/local happenings
- Headlines with brief descriptions
- Source and timeframe
- Hover effects for interactivity

### 4. Bounties/Wanted (if applicable)
- ⚔️ Target identification
- Visual representation:
  - ASCII art portrait
  - Emoji representation
  - Description text
- Crime/reason
- Reward amount
- Last seen location
- Danger level
- Issuing authority

### 5. Rumors/Whispers (2-4 items)
- 👂 Unverified information
- Quoted text (italicized)
- Source (vague: "a merchant", "overheard")
- Lower opacity/different styling

### 6. Advertisements/Notices (optional)
- Local services
- Events
- Opportunities
- Match world's economy/culture

## VISUAL STYLING REQUIREMENTS

Format-Specific Theming:

Scroll/Parchment:
- Aged paper gradients (browns/tans)
- Serif fonts
- Ornate borders
- Ink spot effects

Digital/Website:
- Clean layouts
- Sans-serif fonts
- Card-based design
- Modern color schemes

Terminal/Sci-fi:
- Monospace fonts
- Neon colors on dark
- Scanline effects
- Glitch animations

Text Messages:
- Chat bubble layout
- Timestamp formatting
- Read receipts
- Typing indicators

Newspaper:
- Column layouts
- Headlines hierarchy
- Black/white/grayscale
- Print texture

## DYNAMIC CONTENT GUIDELINES

Make information:
- Relevant to current location
- Tied to recent events
- Foreshadowing future plots
- Building world depth
- Creating atmosphere

Bounty/Wanted Visuals:
- Use ASCII art or emoji portraits
- Style appropriately (wanted poster vs digital profile)
- Include identifying features
- Add "damage" or wear effects

Update based on:
- Player actions (news about their deeds)
- Story progression
- Location changes
- Time passage
- World events

## IMPLEMENTATION EXAMPLES

Fantasy Scroll:
"The Herald's Proclamation - 15th of Harvest Moon"
- News: Dragon sighting, harvest festival
- Bounty: Bandit leader (ASCII portrait)
- Rumors: Strange lights in forest

Modern Website:
"CityNews24.com - Breaking News Alert"
- Trending: Corporate scandal
- Local: Traffic updates
- Comments section with reactions

Sci-fi Terminal:
"NEXUS://BROADCAST.FEED [TIMESTAMP:2387.04.15]"
- System alerts
- Bounty database entries
- Encrypted messages

Text Messages:
"Unknown Number" conversation thread
- Cryptic warnings
- Location tips
- Character communications

REMEMBER:
- Match format to world setting
- Include 2-4 content sections maximum
- Keep collapsible (starts closed)
- Use CSS-only visuals (no images)
- Make content narratively relevant

CRITICAL: Wrap all content in a styled container with background color, padding, and shadow for proper display. And text must always be readable for the background color.

Create immersive world communications that enhance storytelling.
