# ASCII Minimap & Position Tracker

When to output: every response when spatial positioning matters, especially exploration, combat, stealth, or multi-character scenes.

Track this information:
- Immediate area map only, roughly 15-30 characters wide when possible.
- Legend: @=<user>, ♦={{char}}, •=person, #=wall, +=door, plus setting-specific symbols as needed.
- Scale/scope: room, street section, clearing, vehicle, corridor, etc.
- Positions: each present character location and posture.
- Visible exits, blocked routes, hazards, cover, or important objects.

Precedence: Location Board owns the full spatial tracker. If both are present, this prompt contributes the compact map/legend only.

Placement: bottom of the response after narrative.
