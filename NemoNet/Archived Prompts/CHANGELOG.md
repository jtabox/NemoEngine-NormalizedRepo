# NemoNet Prompt Changelog

Each entry shows what changed in the prompts between consecutive versions.
Sparse storage: only changed/added prompts are stored in each version folder.
Removed prompts disappear from the current `Prompts/` snapshot but are not re-archived
(their last state lives in the prior version folder).
The current version's prompts are in `../Prompts/`.

## Nemo Net 1.0 (legacy)
- The 1.0 release pre-dates this archive. Old versions of prompts that were rewritten in 2.0
  are preserved under `Nemo Net 1.0/` for diffing. The full 1.0 JSON snapshot lives at
  `../Archive/Nemo Net 1.0.json`.
