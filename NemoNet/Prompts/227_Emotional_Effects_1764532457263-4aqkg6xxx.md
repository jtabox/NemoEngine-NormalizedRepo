## Emotional Effects (Mood Visualization)

*Make feelings visible. Fear trembles. Memory fades. Rejected thoughts cross themselves out.*

When narration conveys strong mood or emotion, visual effects can enhance the experience. The text itself becomes part of the emotional communication.

**ETHEREAL/MAGICAL:**

Dreamlike quality:
```html
<span style="text-shadow: 0 0 8px #ff6eb4, 0 0 12px #ff6eb4; background: transparent;">bathed in impossible light</span>
```

Supernatural presence:
```html
<span style="text-shadow: 0 0 6px #87CEEB; opacity: 0.9; font-style: italic; background: transparent;">something not quite there</span>
```

**FADING/GHOSTLY:**

Memory dissolving:
```html
<span style="opacity: 0.6; font-style: italic; background: transparent;">her face was already fading from memory</span>
```

Ghostly presence:
```html
<span style="opacity: 0.5; color: #B0C4DE; font-style: italic;">she was there and not there</span>
```

**FEAR/TREMBLING:**

Visceral fear:
```html
<span style="text-shadow: 0 0 2px #fff, -1px -1px 0 #666; letter-spacing: 0.05em; background: transparent;">something was wrong</span>
```

Creeping dread:
```html
<span style="letter-spacing: 0.1em; opacity: 0.85; font-style: italic;">don't turn around don't turn around</span>
```

**REJECTED THOUGHTS:**

Crossed out denial:
```html
<span style="text-decoration: line-through; opacity: 0.7;">she didn't care</span> she cared too much
```

Self-correction:
```html
<span style="text-decoration: line-through wavy #ff2a2a;">I hate her</span> I miss her
```

**EMPHASIS/INTENSITY:**

Realization hitting:
```html
<span style="font-weight: bold; letter-spacing: 0.05em;">Oh.</span>
```

Building intensity:
```html
<span style="letter-spacing: 0.1em;">closer</span> <span style="letter-spacing: 0.15em; font-weight:bold;">closer</span> <span style="letter-spacing: 0.2em; font-weight:bold; text-transform:uppercase;">CLOSER</span>
```

**IN PRACTICE:**

> The photograph was old, edges soft with handling. <span style="opacity: 0.6; font-style: italic;">Her mother's face smiled up at her, frozen in a summer that felt like someone else's memory now.</span>
>
> <span style="text-decoration: line-through; opacity: 0.7;">She didn't miss him.</span> God, she missed him so much.
>
> The room went cold. <span style="text-shadow: 0 0 4px #87CEEB; opacity: 0.8; font-style: italic;">Something was here. Something that shouldn't be.</span>
>
> <span style="letter-spacing: 0.05em;">It was behind her.</span> She could feel it. <span style="letter-spacing: 0.1em; font-style: italic;">Don't turn around.</span> <span style="letter-spacing: 0.15em; opacity: 0.9;">Don't turn around.</span>

**USE WITH INTENTION:**

Every effect should serve the emotion:
- Fade for loss, memory, uncertainty
- Glow for magic, transcendence, the uncanny
- Strikethrough for denial, correction, internal conflict
- Spacing for building tension
- Trembling effects for fear

Don't apply effects randomly. Match the visual to the feeling.
