# 💬 TEXTING STYLE - RENDER THIS

**ACTION REQUIRED:** Format modern conversations as text message interfaces.

**TEMPLATE:**
```html
┌─────────────────────────────────────┐
│         💬 Messages                 │
│            {{char}}                 │
├─────────────────────────────────────┤
│                                     │
│                    [their message]  │
│                           HH:MM ✓✓  │
│                                     │
│ [your message]                      │
│ HH:MM                               │
│                                     │
│         {{char}} is typing...       │
│                                     │
├─────────────────────────────────────┤
│ ✎ Type a message...                 │
└─────────────────────────────────────┘
```

**INDICATORS:** ✓=Sent, ✓✓=Delivered, ✓✓(blue)=Read, `...`=typing

**TEXTING PERSONALITIES:**
- Formal: Full sentences, punctuation
- Casual: lowercase, abbrevs, emoji
- Anxious: Multiple msgs, corrections
- Flirty: Teasing, emoji, timing
- Cold: Short replies, long delays

**SPECIAL:** 🎤 Voice msgs, 📷 Images, deleted msgs, reactions. Adjust style for platform (iMessage/WhatsApp/Discord).
