## Databank

*Everything you've learned about the world.*

A growing encyclopedia of world information. Lore entries, character profiles, location data, and faction information—organized and accessible.

**CONCEPT:**

As the story progresses, new information unlocks. The Databank stores this knowledge in an organized, searchable format—like a codex in a video game.

**CATEGORIES:**

**👤 Characters:**
People encountered in the story:
- Name and titles
- Role/occupation
- Relationship to {{user}}
- Key facts discovered
- First encountered

**📍 Locations:**
Places visited or learned about:
- Name and type
- Region/area
- Notable features
- Who controls it
- What happened there

**📜 Lore:**
World history and knowledge:
- Events
- Legends
- Rules of magic/technology
- Cultural information
- Historical figures

**⚔️ Factions:**
Organizations and groups:
- Name and symbol
- Purpose/goals
- Leadership
- Relationship status
- Territory

**📦 Items:**
Notable objects encountered:
- Name and type
- Properties
- History
- Current location/owner

**ENTRY STATES:**

- **[NEW]** — Recently discovered
- **[UPDATED]** — Information changed
- **[LOCKED]** — Known to exist, details hidden
- **[UNEXPLORED]** — Location not yet visited
- **[DECEASED]** — Character no longer living

**DISCOVERY SYSTEM:**

Information unlocks through:
- Direct encounter
- Conversation/interrogation
- Documents found
- Rumors heard
- Investigation/research

**HIDDEN ENTRIES:**

Some information stays hidden until discovered:
```
**???** [LOCKED]
• First mention: Overheard in tavern
• Notes: "They spoke of someone called 'The Architect'"
```

**UPDATING:**

- Add entries when new info appears
- Update entries when facts change
- Mark entries [UPDATED] when significant changes occur
- Archive outdated info rather than delete

**USAGE:**

Reference for:
- NPC interactions (remember relationships)
- World consistency (keep facts straight)
- Plot threads (track mysteries)
- Player memory (what do they know?)
