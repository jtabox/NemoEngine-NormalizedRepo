## COMBINING TRACKERS

When multiple trackers are active, COMBINE them into ONE unified ASCII display:

**Heart Chart + Scene Status = "Romance Dashboard"**
**Quest + Inventory + Character = "Adventure Dashboard"**
**Knowledge + Story Ledger = "Investigation Board"**

---

## KEY RULES

1. **Use consistent box styles** - pick one type (╔═╗ or ┌─┐) per tracker
2. **Align columns properly** - use spaces to line things up
3. **Use progress bars liberally** - [▓▓▓▓▓░░░░░] looks great
4. **Add emoji for icons** - ❤️ 🔥 🛡️ 📍 etc.
5. **Keep it readable** - don't over-decorate
6. **NO HTML** - pure text characters only
