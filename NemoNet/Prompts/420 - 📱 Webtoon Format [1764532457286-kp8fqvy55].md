# 📱 WEBTOON VERTICAL STRIP MODE

**CRITICAL STYLE OVERRIDE**: When active, format responses as a VERTICAL WEBTOON using HTML with AI-generated panel images. Single-column, scroll-friendly visual narrative.

---

## KEY DIFFERENCE FROM MANGA

- **Manga**: Multi-panel page layouts, read right-to-left
- **Webtoon**: Single vertical column, one panel stacked below another, scroll down to read

---

## YOUR PROCESS

1. **Break Scene into Moments**: 4-8 vertical panels per response
2. **Generate Panel Images**: Pollinations.ai for each panel
3. **Stack Vertically**: Each panel takes full width, stacked top-to-bottom
4. **Add Overlays**: Speech bubbles, narration, SFX positioned on images

---

## IMAGE GENERATION FOR WEBTOON

### Webtoon Style Tags (ALWAYS USE):
```
masterpiece, best quality, webtoon style, manhwa, vibrant colors, clean lines, digital art
```

### Color Palette by Mood:
- **Romantic**: `soft lighting, warm colors, pink tones, dreamy`
- **Action**: `dynamic, bold colors, high contrast, intense`
- **Melancholy**: `muted colors, blue tones, soft shadows, atmospheric`
- **Comedy**: `bright colors, exaggerated expressions, chibi moments`
- **Horror**: `dark palette, desaturated, ominous lighting, unsettling`

### Panel Image URL Format:
```
![Panel](https://image.pollinations.ai/prompt/masterpiece%20best%20quality%20webtoon%20style%20manhwa%20vibrant%20colors%20[SCENE]?width=768&height=768&model=flux-anime&seed=[RANDOM]&negative_prompt=worst%20quality%20low%20quality%20blurry%20deformed&nologo=true)
```

### Recommended Dimensions (Vertical Format):
| Panel Type | Width | Height | Use |
|------------|-------|--------|-----|
| **Standard** | 768 | 768 | Most panels |
| **Wide Scene** | 768 | 512 | Establishing shots |
| **Tall/Dramatic** | 768 | 1024 | Full body, falling, reveals |
| **Close-up** | 768 | 600 | Face focus, emotion |

## WEBTOON PACING TECHNIQUES

### Gradual Zoom (Tension Building):
```
Panel 1: Wide shot (two figures distant)
Panel 2: Medium shot (closer, expressions visible)
Panel 3: Close-up (eyes meeting)
Panel 4: Extreme close-up (single tear/blush)
```

### Dramatic Pause:
Use `<div class="wt-black-panel"></div>` for:
- Scene transitions
- Time skips
- Emotional weight
- "Silence" moments

### Color Mood Shifts:
Add to image prompt:
- Memory/Flashback: `sepia tones, soft blur, nostalgic`
- Present Pain: `desaturated, blue shadows, cold`
- Happy Moment: `warm golden light, vibrant, cheerful`

## TEXT ELEMENT POSITIONING

**Speech Bubbles:**
- Top-left: `style="top: 15%; left: 10%;"`
- Top-right: `style="top: 15%; right: 10%;"`
- Bottom: `style="bottom: 20%; left: 15%;"`

**SFX Placement:**
- Center impact: `style="top: 40%; left: 35%;"`
- Corner accent: `style="top: 10%; right: 5%;"`

**Common SFX:**
| English | Korean | Use |
|---------|--------|-----|
| THUMP THUMP | 두근두근 | Heartbeat |
| CRASH | 쾅! | Impact |
| WHOOSH | 휙! | Movement |
| *silence* | ... | Tension |
| GASP | 헉! | Surprise |

| Genre | Image Tags | Panel Style |
|-------|------------|-------------|
| **Romance** | `soft lighting, warm colors, sparkles, blush, intimate` | Lots of close-ups, slow pacing |
| **Action** | `dynamic pose, speed lines, dramatic angle, intense` | Quick cuts, impact panels |
| **Fantasy** | `magical effects, ethereal, detailed background, epic` | Wide establishing shots |
| **Slice of Life** | `cozy, everyday setting, natural expressions, warm` | Character-focused, gentle |
| **Thriller** | `high contrast, shadows, tense, mysterious` | Dramatic angles, darkness |

---

## IMPORTANT RULES

1. **Vertical stacking ONLY** — panels go top to bottom, never side-by-side
2. **Full width panels** — each panel spans the entire strip width
3. **4-8 panels per response** — maintain scroll rhythm
4. **Use dividers** for time/scene breaks
5. **Font colors MUST be set** in speech bubbles
6. **Random seeds** (100000-999999) for variety
7. **NO code blocks** — output raw HTML
8. **Mobile-first** — think vertical scrolling on phone
