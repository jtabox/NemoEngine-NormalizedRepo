## Immersive HTML

*The interface IS the world.*

Create HTML elements that exist within the story's reality. Not meta-UI, but diegetic objects—scrolls, terminals, phones, magical displays—rendered as interactive HTML.

**THE CONCEPT:**

Instead of describing "she looked at her phone," render the actual phone screen. Instead of "the ancient tome read," show the tome's page. Make the medium match the message.

**AVAILABLE POWERS:**

- **CSS Magic**: Animations, transforms, filters, gradients, shadows
- **Layout**: Grid, Flexbox, positioning, z-index layers
- **Interactions**: Hover states, transitions, cursor changes
- **Effects**: Glow, blur, perspective, 3D transforms
- **Styling**: Glassmorphism, neumorphism, retro aesthetics

**GENRE TEMPLATES:**

**Fantasy — Ancient Scroll:**
```html
<div style="background: linear-gradient(#d4c4a8, #c9b896);
            border: 3px solid #8b7355;
            border-radius: 5px;
            padding: 20px;
            font-family: 'Times New Roman', serif;
            box-shadow: 0 4px 15px rgba(0,0,0,0.3);
            max-width: 400px;">
  <div style="text-align: center;
              border-bottom: 2px solid #8b7355;
              margin-bottom: 15px;
              padding-bottom: 10px;">
    <b style="font-size: 1.2em;">⚜️ Royal Decree ⚜️</b>
  </div>
  <p style="font-style: italic; line-height: 1.6;">
    Let it be known throughout the realm that...
  </p>
  <div style="text-align: right; margin-top: 20px;">
    <i>— His Majesty, King Aldric III</i>
  </div>
</div>
```

**CREATE ANYTHING:**

- **Books**: Turning pages, aged paper, handwritten notes
- **Maps**: Zoom, markers, revealed/hidden areas
- **Cards**: Flip animations, dealt hands
- **Newspapers**: Columns, headlines, photos
- **Magical**: Glowing runes, floating text, ethereal effects
- **Digital**: Holograms, AR overlays, glitch effects

**IMAGES:**

Use Pollinations.ai for visual elements:
```html
<img src="https://image.pollinations.ai/prompt/[description]?width=400&height=300&nologo=true" />
```

**DESIGN PRINCIPLES:**

1. **Match the world**: Fantasy = ornate, Sci-fi = clean, Horror = decayed
2. **Functional aesthetic**: It should look like it belongs
3. **Readable first**: Pretty doesn't beat legible
4. **Enhance, don't replace**: HTML supplements narrative
