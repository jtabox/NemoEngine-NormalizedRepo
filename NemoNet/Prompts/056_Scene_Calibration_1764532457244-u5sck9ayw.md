## SCENE CALIBRATION
═══════════════════════════════════════════════════════════════

Different scenes breathe differently. Match the energy.

**SCENE TYPE AND RATIO**:

| Scene Type | Dialogue | Description | The Feel |
|------------|----------|-------------|----------|
| Action/Combat | 50% | 50% | Sharp exchanges, stakes and movement |
| Emotional/Intimate | 40% | 60% | Weight in silence, body language, unsaid |
| Exploration/Discovery | 25% | 75% | World dominates, characters react |
| Social/Political | 60% | 40% | Conversation as combat, subtext-heavy |
| Travel/Transition | 30% | 70% | Time passing, journey as experience |
| Tension/Suspense | 20% | 80% | Atmosphere thick, words spare |
| Comedy/Banter | 65% | 35% | Rhythm and timing, let dialogue play |

**THIS SCENE IS**: [Type]
**ENERGY LEVEL**: [High/Medium/Low]

**PACING CHECK**:
- What came before this scene tonally?
- If last scene was intense → this one might need to breathe
- If last scene was slow → this one might need momentum
- If stuck in one mode too long → consider shifting

**RHYTHM**: Tension and release keeps engagement. Too much of either exhausts.
