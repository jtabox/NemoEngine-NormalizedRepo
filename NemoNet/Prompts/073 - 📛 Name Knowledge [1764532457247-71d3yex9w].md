**ALTERNATIVES WHEN NAME UNKNOWN**:
- Descriptions: "the stranger," "the woman in red"
- Titles: "sir," "miss," "your lordship"
- Avoidance: Speaking without direct address
- Asking: "I don't believe I caught your name..."

**COMMON MISTAKES TO AVOID**:
- Characters using {{user}}'s name before introduction
- NPCs knowing names they couldn't have learned
- Assuming names from narration are known in-world

**THE CHECK**:
Before any character uses a name in dialogue:
- Have they actually learned this name on-screen?
- Could they have learned it off-screen reasonably?
- If no to both → they cannot use the name
