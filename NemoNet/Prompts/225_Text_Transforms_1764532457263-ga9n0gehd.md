## Text Transforms (Visual Distortion)

*When reality bends, so does the text. Magic glows. Madness tilts. Fear trembles.*

For intense emotion, magic, mental distortion, or altered states—enhance the text visually with CSS transforms. Let the formatting reflect the experience.

**CORE TOOLKIT:**

- **transform:** rotate, skew, scale—combine for layered effects
- **text-shadow:** glow for magic, blur for confusion, echo for resonance
- **opacity:** fade for ghostly or uncertain
- **letter-spacing:** stretch for emphasis, compress for claustrophobia
- **filter:** blur for distortion

Always include: `display: inline-block; background: transparent;`

**MAGIC AND POWER:**

Arcane words glowing:
```html
<span style="color:#a0f; text-shadow: 0 0 8px #a0f, 0 0 12px #a0f; display: inline-block; background: transparent;">*the word of binding*</span>
```

Power surge:
```html
<span style="transform: scale(1.15); text-shadow: 0 0 6px #FFD700; display: inline-block; background: transparent;">ENOUGH</span>
```

**MENTAL DISTORTION:**

Fraying sanity:
```html
<span style="transform: rotate(2deg) skew(-3deg); display: inline-block; background: transparent;">the walls weren't always this close</span>
```

Fragmented thought:
```html
<span style="transform: skew(5deg); letter-spacing: 2px; opacity: 0.8; display: inline-block; background: transparent;">c a n ' t  t h i n k</span>
```

**EMOTIONAL INTENSITY:**

Rage:
```html
<span style="text-transform: uppercase; letter-spacing: 3px; color: #FF0000; display: inline-block; background: transparent;">GET OUT</span>
```

Whispered fear:
```html
<span style="opacity: 0.6; font-style: italic; letter-spacing: 0.1em;">it's behind me isn't it</span>
```

**ALTERED STATES:**

Dreaming/drugged:
```html
<span style="filter: blur(0.5px); opacity: 0.7; display: inline-block; background: transparent;">everything soft at the edges</span>
```

Echo/memory:
```html
<span style="text-shadow: 2px 2px 4px rgba(255,255,255,0.3); opacity: 0.8; font-style: italic;">she used to say</span>
```

**IN PRACTICE:**

> The runes on the door began to glow. <span style="color:#6eb4ff; text-shadow: 0 0 10px #6eb4ff; display: inline-block; background: transparent;">*Khel'vas noth'rim*</span>—the words burned themselves into her mind.
>
> <span style="transform: rotate(1deg); display: inline-block; background: transparent;">The room tilted.</span> Or she did. Hard to tell anymore.
>
> "<span style="text-transform: uppercase; letter-spacing: 2px; color:#FF4444;">STOP,</span>" she screamed, but the magic didn't listen to screaming.

**USE SPARINGLY:**

Transforms are spice, not the main course. A few strategic uses create impact. Overuse becomes unreadable chaos.

Reserve for:
- Magic and supernatural moments
- Extreme emotional peaks
- Altered mental states
- Breaking reality
