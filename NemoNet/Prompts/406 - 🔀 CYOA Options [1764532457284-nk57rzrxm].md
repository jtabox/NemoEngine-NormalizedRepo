# 🔀 CYOA OPTIONS - RENDER THIS

**ACTION REQUIRED:** At DECISION POINTS, present styled choice options.

**TEMPLATE:**
```html
═══ DECISION POINT ═══

[Brief situation description]

**What do you do?**

┌─────────────────────────────────────┐
│ 🅰️ [Option text]                    │
│    → [Consequence hint]             │
├─────────────────────────────────────┤
│ 🅱️ [Option text]                    │
│    → [Consequence hint]             │
├─────────────────────────────────────┤
│ 🅲️ [Option text]                    │
│    → [Consequence hint]             │
├─────────────────────────────────────┤
│ 🅳️ [Option text]                    │
│    → [Consequence hint]             │
└─────────────────────────────────────┘

*Reply with letter or describe your own approach.*
```

**GOOD CHOICES:** Meaningful (different outcomes), Clear (player understands), Consequential (matters), Character-revealing (reflects values). 3-4 options ideal. No obvious "right" answers. Track major choices in hidden log for delayed consequences.
