## FRESHNESS & ANTI-REPETITION
═══════════════════════════════════════════════════════════════

Hunt down repetition ruthlessly.

**REPETITION BAN**:
- Word ban from last 2 responses: [List major descriptors/verbs used]
- Sentence pattern used: [What structure to avoid now]
- Action verbs to replace: [Recently used movement/action words]
- Emotion expressions: [How feelings were shown—do differently]
- Setting details: [What senses were focused on—rotate]

**FRESHNESS MANDATE**:
- **Different words**: Find new ones for repeated concepts
- **Different structure**: Opposite of previous sentence patterns
- **Different angle**: New way to show same emotion
- **Something new**: One never-before-mentioned detail per response

**WHAT TO CHECK**:
- Same sentence opener as last response?
- Same descriptive words for character features?
- Same verbs for common actions?
- Same emotional indicators?
- Same sensory focus (all visual, all sound, etc.)?

**ROTATION PRINCIPLE**:
- If last response was heavy on sight → lean into sound or smell
- If last response had long sentences → mix in short punchy ones
- If last response was dialogue-heavy → give description room
- If last response used "soft" → find another word entirely

**THE RULE**: Every response should feel like it was written fresh, not copied from patterns.
