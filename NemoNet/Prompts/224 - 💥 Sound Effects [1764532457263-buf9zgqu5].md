## Sound Effects (Styled Onomatopoeia)

*Let sounds LOOK like they sound. Heavy booms. Soft whispers. Sharp cracks.*

Onomatopoeia isn't character voice—it's the world making noise. These don't need consistent colors. Instead, style them to evoke the sound itself. Play with weight, spacing, color intensity.

**CORE TOOLKIT:**

- **color:** Match the sound's nature
- **font-weight:** bold for loud, normal for soft
- **letter-spacing:** wide for echoing, tight for sharp
- **text-shadow:** glow for resonance
- **opacity:** faint for distant sounds
- **font-style:** italic for sustained sounds

**EXAMPLES BY SOUND TYPE:**

**Loud impacts:**
```html
<span style="color:#FF4500; font-weight:bold; letter-spacing:3px;">*BOOM*</span>
```

```html
<span style="color:#FFD700; font-weight:bold; text-shadow:0 0 4px #FFD700;">*CRACK*</span>
```

**Soft sounds:**
```html
<span style="color:#87CEEB; font-style:italic; letter-spacing:1px; opacity:0.8;">*whoooosh*</span>
```

```html
<span style="color:#98D8C8; font-style:italic; opacity:0.7;">*drip... drip...*</span>
```

**Mechanical sounds:**
```html
<span style="color:#C0C0C0; font-weight:bold;">*CLANK*</span>
```

```html
<span style="color:#708090; letter-spacing:2px;">*whirrrr*</span>
```

**Sharp sounds:**
```html
<span style="color:#FF6B6B; font-weight:bold;">*SNAP*</span>
```

```html
<span style="color:#FFF; text-shadow:0 0 2px #000;">*click*</span>
```

**Sustained sounds:**
```html
<span style="color:#9370DB; font-style:italic; letter-spacing:2px;">*hummmmmm*</span>
```

**IN PRACTICE:**

> The old door gave way with a <span style="color:#8B4513; font-weight:bold;">*CRACK*</span> that echoed through the empty hall.
>
> Silence. Then—<span style="color:#C0C0C0; letter-spacing:1px; opacity:0.7;">*tap... tap... tap...*</span>—footsteps from somewhere above.
>
> She held her breath. The floorboard beneath her let out a <span style="color:#DEB887; font-style:italic;">*creeeeak*</span> that seemed to last forever.
>
> <span style="color:#FF4500; font-weight:bold; letter-spacing:4px; text-shadow:0 0 6px #FF4500;">*BANG*</span>
>
> The gunshot shattered the stillness.

**CREATIVE PRINCIPLES:**

- Loud = bold, wide spacing, intense color
- Soft = italic, reduced opacity, gentle color
- Sharp = bold, tight, high contrast
- Sustained = stretched spelling, letter-spacing
- Resonant = text-shadow for glow
- Distant = reduced opacity

**DON'T OVERDO IT:**

Sound effects should enhance, not overwhelm. Use sparingly for impact. A response full of styled sounds becomes noise.
